# DGS-SparseSampling

This repository contains the MATLAB source code for the numerical experiments presented in our paper on **On Random Sampling of Diffused Graph Signals with Sparse Inputs on Vertex Domain**. The code evaluates the sampling and recovery performance of sparse signals across various graph topologies, and sampling strategies (Uniform Random vs. Variable Density).

##  Dependencies

To run the scripts in this repository, you will need:

- **MATLAB** (Tested on recent versions, requires the Optimization Toolbox for `linprog`).
- **GSPBox (Graph Signal Processing Toolbox)**.

------

##  Repository Structure & Experiments

The repository is organized into distinct scripts corresponding to the numerical examples in the paper. All signal recovery tasks are formulated as $L_1$-minimization problems (Basis Pursuit/BPDN) .

### Example I: Performance Evaluation Under Varying Noise Levels

Evaluates the sampling performance of binary diffusion models across four graph topologies: Regular, Erdös-Rényi (ER), Small-World, and Stochastic Block Model (SBM).

- `example_I_test.m`: Compares the noiseless recovery performance across the four topologies, demonstrating the structural dependence of the incoherence parameter $\mu$ and the sparse condition number $\kappa(\Gamma)$.
- `phase_test.m`: Generates the empirical phase transition diagram for exact recovery on ER random graphs, verifying the theoretical scaling trend $m \propto k$.
- `noise_test.m`: Evaluates robust recovery under noisy observation models ($\|y - H_{\mathcal{M}}\alpha\|_2 \le \eta$) with varying Signal-to-Noise Ratios (SNR: 10 dB to 40 dB).

### Example II: ER and Small-World Graphs (Binary Diffusion)

Investigates the relationship between the required number of samples and the internal structural probabilities of specific graph models.

- `example_II_ER.m`: Tests large-scale ER random graphs (N = 10000) under different connection probabilities $b$. Highlights how the network density variance dictates the sample bounds.
- `example_II_WS.m`: Evaluates small-world graphs under varying rewiring probabilities $b$. Shows the convergence behavior of both the sparse input $\alpha$ and the diffused signal $x$.

### Example III: Variable Density Sampling

Examines the proposed variable density sampling strategy against uniform random sampling.

- `example_III.m`: Runs on diffusion models generated by strictly Doubly Stochastic matrices. Demonstrates that variable density sampling exploits subtle local coherence variations to minimize reconstruction error.

### Example IV: Real-World Source Localization

Applies the theoretical framework to a practical urban monitoring scenario.

- `example_IV.m`: Models the dispersion of a chemical pollutant on the **Minnesota Road Network** (N = 2642). Demonstrates that the deterministic variable density sampling strategy strategically places sensors at topological hubs, outperforming uniform random sampling.
